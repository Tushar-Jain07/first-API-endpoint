# First API Endpoint

My first API built with Node.js and Express as part of the FlyRank internship.

## How to run

```
npm install
npm start
```

Server runs on http://localhost:3000

## Endpoints

**GET /api/info** - Returns project info

**GET /api/ping** - Health check endpoint

## Testing with curl

```
curl http://localhost:3000/api/info
curl http://localhost:3000/api/ping
```
